## 🐛 Troubleshooting

### Images Tidak Muncul?
- Cek koneksi internet (menggunakan Unsplash)
- Restart dev server
- Clear browser cache

### Section Tidak Terlihat?
- Cek `app/page.tsx` - pastikan import sudah benar
- Cek console untuk errors
- Verify component names

### Active State Navbar?
- Pastikan section ID match dengan navLinks
- Scroll ke section untuk test
- Cek browser console
