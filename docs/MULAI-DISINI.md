```bash

```

## 🌐 Deploy ke Internet (10 Menit)

### Cara Termudah: Vercel (Gratis)

1. Push ke GitHub:
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/username/elfan-landing.git
git push -u origin main
```

2. Deploy:
- Buka [vercel.com](https://vercel.com)
- Sign in dengan GitHub
- Import repository
- Click "Deploy"

**Selesai!** Website sudah online.

📖 **Detail lengkap**: Baca `DEPLOYMENT.md`

## 🆘 Butuh Bantuan?

### Troubleshooting Umum:

**Error saat install?**
```bash
rm -rf node_modules
npm install
```

**Port 3000 sudah dipakai?**
```bash
npm run dev -- -p 3001
```

**Gambar tidak muncul?**
- Restart server: `Ctrl+C` lalu `npm run dev`
- Cek path gambar: harus `/images/file.jpg`
